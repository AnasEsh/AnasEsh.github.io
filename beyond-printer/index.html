Text

Printing text and related options listed in this section

Text Format

Text Format is a utility class used to format printed text, it must be provided to text printing methods.

val format= TextFormat();
format.fontSize=35;
format.align= PrintAlign.FORMAT_ALIGN_CENTER;
format.isBold=true;

printText(TextFormat format, String text) → void

Used to add line of text to print, given the format and string.

Example
printText(defaultTextFormat, "Example String")

printMultiText(TextFormat format, String left, String right) → void

Used to add left and right aligned text in the same line.

Example
printMultiText(defaultTextFormat, "left Str","right Str")

Bitmap

This section describes the format and commonly used methods to print bitmaps

Bitmap Format

utility class that can be used to configure the format in which the image will be printed, like width, height, alignment and more.

val bitmapFormat=BitmapFormat();
bitmapFormat.align=PrintAlign.FORMAT_ALIGN_CENTER;
bitmapFormat.width=(image.width*1.1).toInt();
bitmapFormat.height=(image.height*1.1).toInt();

printBitmap(BitmapFormat format, Bitmap bitmapImage) → void

Used to print bitmap images, given the bitmap format and the image itself

Example
POSDeviceManager.getInstance().printerDevice.apply {
val format=BitmapFormat();
format.align=PrintAlign.FORMAT_ALIGN_CENTER;
format.width=(image.width*1.1).toInt();
format.height=(image.height*1.1).toInt();
printText(defaultTextFormat,"\n");
printBitmap(format,image);
cutPaper();
startPrint(onPrintListener);
}

Barcode

Used to print any barcode, like product barcodes, supporting all common barcode formats.

BarCode Format

Formats the printed barcodes, configures which barcode type to use, used Human Readable Interpretation(HRI), width, height and alignment. 

val codeFormat=BarCodeFormat();
codeFormat.barCodeType=PrintBarCodeType.BARCODE_TYPE_CODE128;
codeFormat.align=PrintAlign.FORMAT_ALIGN_CENTER;
codeFormat.hriLocation=PrintBarCodeHriLocation.FORMAT_BARCODE_HRI_LOCATION_DOWN;
codeFormat.height=180;
codeFormat.width=300;

printBarCode(BarCodeFormat format, String code) → void

Used to print barcodes with the specified format and code value

Example 
POSDeviceManager.getInstance().printerDevice.apply {
	val format=BarCodeFormat();
	format.barCodeType=PrintBarCodeType.BARCODE_TYPE_CODE128;
	format.align=PrintAlign.FORMAT_ALIGN_CENTER;
	format.hriLocation=PrintBarCodeHriLocation.FORMAT_BARCODE_HRI_LOCATION_DOWN;
	printBarCode(format,productCode);
	printText(defaultTextFormat,"\n \n \n")
	cutPaper();
	startPrint(onPrintListener);
}

Qr Codes

QR codes can encode a variety of data types including URLs, phone numbers, and other strings.
For the best results, specify the appropriate resolver scheme, such as https:// for web links or tel: for phone numbers, mailto ...
Common Schemes

QrCode Format

Used for general formatting of the printed QR codes. 

val qrFormat=QrCodeFormat();
qrFormat.height=200;
qrFormat.width=200;
qrFormat.align=PrintAlign.FORMAT_ALIGN_CENTER;
(qrFormat,text)

printQrCode(QrCodeFormat format, String content) → void

Used to print qr code with the specified format and content value

Example 
POSDeviceManager.getInstance().printerDevice.apply {
val qrFormat=QrCodeFormat();
qrFormat.height=200;
qrFormat.width=200;
qrFormat.align=PrintAlign.FORMAT_ALIGN_CENTER;
printQrCode(qrFormat,"https://beyond.com.sa/");
printText(defaultTextFormat,"\n\n\n");
cutPaper();
startPrint(onPrintListener);
}

Common methods

This section describes common functionalities needed while printing any of the supported formats
print result is passed as a callback when sending print request using startPrint(listener) which is required to send any printing request

Abstract OnPrintTaskListener()

Abstract class with one member method, onPrintResult(resultCode)

Example 
val onPrintListener=object: OnPrintTaskListener(){
override fun onPrintResult(p0: Int) {
	when(p0){
		PrinterConstants.TASK_STATUS_ERR -> Log.e("Print result", "Error");
		PrinterConstants.TASK_STATUS_CANCEL -> Log.i("Print result", "Cancelled");
		PrinterConstants.TASK_STATUS_FAIL -> Log.e("Print result", "Failed");
		PrinterConstants.TASK_STATUS_SUCCESS -> Log.i("Print result", "Success");
		PrinterConstants.TASK_STATUS_OUT_OF_PAPER -> Log.e("Print result", "Out of paper");
		PrinterConstants.TASK_STATUS_OVER_HEAT -> Log.e("Print result", "Overheated");
		PrinterConstants.TASK_STATUS_LOW_POWER -> Log.e("Print result", "Low dev power");
		else ->Log.i("Print result","Unknown state");
	}
	Log.i("Print result int is",p0.toString());
}
};

startPrint(OnPrintTaskListener listener) → void

Mandatory to send and execute any printing request

Example 
POSDeviceManager.getInstance().printerDevice.apply {
startPrint(onPrintListener);
}

ESC commands

ESC (Escape) commands refer to a set of printer control codes that use the Escape character to trigger special functionality. These commands allow you to access advanced printer features and perform low-level control operations, such as:

  • Changing printer settings like font, text size, and orientation
  • Enabling or disabling specific printer features
  • Sending direct control sequences to the printer
  • Querying printer status and retrieving diagnostic information
  • Managing paper handling and cutting
  • Performing printer calibration and maintenance

More

  • Use sendESCCommand(byte[] esc)
  • ESC standard documentation
In most cases you dont need to deal with plain ESC commands, and depending on the used framework you will find a library that helps by providing high level methods that handles ESC commands for you
